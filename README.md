Neon Sudoku — Web (Single File)

A sleek, dark-themed Sudoku game in a single HTML file (sudoku.html). No external libraries needed.

View your app in AI Studio: Open App

Quick Start

Save sudoku.html.

Open in any modern browser.

Play using on-screen controls or keyboard shortcuts.

Features

Smart puzzles: Unique-solution boards with three difficulty levels.

Human-style hints: Step-by-step guidance without giving away answers.

Explain mistakes: Highlights errors and explains why moves are invalid.

Real-time conflict detection: Shows duplicates in rows, columns, and boxes instantly.

Digit lock: Prevents entering numbers already completed correctly.

Notes mode: Add candidate numbers; automatically clears conflicting notes.

Undo & history: Undo recent moves, including notes.

Timer & mistakes: Track your solving time and confirmed errors.

Mobile-friendly UI: Responsive 9×9 grid and on-screen number pad.

Controls

UI Buttons: New Game, Hint, Notes, Undo, Check Solution

Keyboard Shortcuts:

1–9: Enter number or toggle note

N: Toggle Notes mode

Backspace/Delete: Clear cell

Ctrl+Z: Undo

Arrow keys: Navigate grid

H: Hint

C: Check solution

R: New Game

Run Locally

Prerequisites: Node.js

Install dependencies:
npm install

Set your Gemini API key in .env.local:
GEMINI_API_KEY=your_key_here

Run the app:
npm run dev
